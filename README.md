# chat_api

A websockets chat API PoC


Requirements:

- Developed with python 3.12 in Ubuntu WSL2


# Install

```
poetry install
```

# Execute

In the root folder, start the poetry venv shell:

```
poetry shell
```

Go to the `src/api` folder and run:

```
fastapi run main.py
```

or

```
fastapi dev main.py
```
